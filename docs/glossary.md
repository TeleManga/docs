## 1. Глоссарий

### 1.1. Общие термины

| Термин | Определение |
|--------|-------------|
| **Манга** | Японские комиксы, основной контент платформы. Состоит из глав, каждая из которых содержит набор страниц-изображений |
| **Глава (Chapter)** | Структурная единица манги. Имеет порядковый номер, может быть привязана к тому. Содержит одну или более страниц |
| **Том (Volume)** | Логическая группировка глав манги. Необязательный атрибут главы |
| **Страница (Page)** | Одно изображение внутри главы. Имеет порядковый номер и URL изображения |
| **Жанр (Genre)** | Классификационная категория манги. Манга может принадлежать нескольким жанрам одновременно |
| **Тег (Tag)** | Дополнительная метка для уточнённой классификации манги. Манга может иметь произвольное количество тегов |
| **Рейтинг (Rating)** | Оценка манги пользователем по шкале от 1 до 10. Каждый пользователь может поставить одну оценку на одну мангу |
| **Комментарий (Comment)** | Текстовый отзыв пользователя к конкретной манге |
| **Библиотека пользователя (User Library)** | Персональный список манги пользователя с присвоенным статусом чтения |
| **История чтения (Reading History)** | Запись прогресса чтения: фиксирует последнюю прочитанную страницу в конкретной главе конкретной манги |

### 1.2. Статусы манги

| Статус | Описание |
|--------|----------|
| **ongoing** | Манга выходит, новые главы публикуются |
| **announced** | Манга анонсирована, главы ещё не вышли |
| **completed** | Публикация манги завершена |
| **dropped** | Публикация манги прекращена (заброшена) |
| **paused** | Публикация манги приостановлена |

### 1.3. Статусы в библиотеке пользователя

| Статус | Описание |
|--------|----------|
| **reading** | Пользователь читает мангу в данный момент |
| **completed** | Пользователь прочитал мангу полностью |
| **plan_to_read** | Пользователь планирует прочитать мангу |
| **dropped** | Пользователь бросил чтение |
| **favorite** | Манга добавлена в избранное |

### 1.4. Роли пользователей

| Роль | Описание |
|------|----------|
| **user** | Обычный пользователь. Может читать мангу, ставить оценки, оставлять комментарии, управлять своей библиотекой |
| **admin** | Администратор. Обладает всеми правами пользователя, а также может добавлять, редактировать и удалять мангу, главы, страницы, жанры и теги |

### 1.5. Технические термины

| Термин | Определение |
|--------|-------------|
| **Telegram Mini App** | Веб-приложение, запускаемое внутри мессенджера Telegram через WebView. Использует Telegram Web App API для взаимодействия с клиентом Telegram |
| **SPA (Single Page Application)** | Одностраничное веб-приложение, которое загружается один раз и динамически обновляет содержимое без перезагрузки страницы |
| **REST API** | Архитектурный стиль взаимодействия клиента и сервера через HTTP-запросы |
| **JWT (JSON Web Token)** | Формат токена для аутентификации и авторизации пользователей |
| **UUID** | Универсальный уникальный идентификатор, используемый в качестве первичного ключа сущностей |
