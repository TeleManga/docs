## 1. Глоссарий

### 1.1. Общие термины

| Термин | Определение |
|--------|-------------|
| **Манга** | Японские комиксы. На платформе представлены наряду с манхвой и маньхуа; структура контента (главы, страницы) едина для всех этих форматов |
| **Манхва (Manhwa)** | Корейские комиксы. Просмотр на платформе реализован так же, как у манги: произведение состоит из глав, глава — из страниц-изображений |
| **Маньхуа (Manhua)** | Китайские комиксы. Просмотр на платформе реализован так же, как у манги: произведение состоит из глав, глава — из страниц-изображений |
| **Глава (Chapter)** | Структурная единица произведения (манги, манхвы, маньхуа). Имеет порядковый номер, может быть привязана к тому. Содержит одну или более страниц |
| **Том (Volume)** | Логическая группировка глав произведения. Необязательный атрибут главы |
| **Страница (Page)** | Одно изображение внутри главы. Имеет порядковый номер и URL изображения |
| **Жанр (Genre)** | Классификационная категория произведения. Произведение может принадлежать нескольким жанрам одновременно |
| **Тег (Tag)** | Дополнительная метка для уточнённой классификации произведения. У произведения может быть произвольное количество тегов |
| **Рейтинг (Rating)** | Оценка произведения пользователем по шкале от 1 до 5. Каждый пользователь может поставить одну оценку на одно произведение |
| **Комментарий (Comment)** | Текстовый отзыв пользователя к конкретному произведению |
| **Библиотека пользователя (User Library)** | Персональный список произведений пользователя с присвоенным статусом чтения |
| **История чтения (Reading History)** | Запись прогресса чтения: фиксирует последнюю прочитанную страницу в конкретной главе конкретного произведения |

### 1.2. Статусы публикации произведения

| Статус | Описание |
|--------|----------|
| **ongoing** | Произведение выходит, новые главы публикуются |
| **announced** | Произведение анонсировано, главы ещё не вышли |
| **completed** | Публикация произведения завершена |
| **dropped** | Публикация произведения прекращена (заброшена) |
| **paused** | Публикация произведения приостановлена |

### 1.3. Статусы в библиотеке пользователя

| Статус | Описание |
|--------|----------|
| **reading** | Пользователь читает произведение в данный момент |
| **completed** | Пользователь прочитал произведение полностью |
| **plan_to_read** | Пользователь планирует прочитать произведение |
| **dropped** | Пользователь бросил чтение |
| **favorite** | Произведение добавлено в избранное |

### 1.4. Роли пользователей

| Роль | Описание |
|------|----------|
| **user** | Обычный пользователь. Может читать произведения (мангу, манхву, маньхуа), ставить оценки, оставлять комментарии, управлять своей библиотекой |
| **admin** | Администратор. Обладает всеми правами пользователя, а также может добавлять, редактировать и удалять произведения, главы, страницы, жанры и теги |

### 1.5. Технические термины

| Термин | Определение |
|--------|-------------|
| **Telegram Mini App** | Веб-приложение, запускаемое внутри мессенджера Telegram через WebView. Использует Telegram Web App API для взаимодействия с клиентом Telegram |
| **SPA (Single Page Application)** | Одностраничное веб-приложение, которое загружается один раз и динамически обновляет содержимое без перезагрузки страницы |
| **REST API** | Архитектурный стиль взаимодействия клиента и сервера через HTTP-запросы |
| **JWT (JSON Web Token)** | Формат токена для аутентификации и авторизации пользователей |
