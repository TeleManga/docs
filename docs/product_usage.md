## 4. Условия эксплуатации

### 4.1. Требования к клиентскому окружению

Приложение доступно в двух режимах, каждый из которых предъявляет свои требования к окружению.

#### 4.1.1. Режим Telegram Mini App

| Платформа | Минимальная версия | WebView-движок |
|-----------|--------------------|----------------|
| **Android** | Telegram 6.9+ | Android System WebView (Chromium) |
| **iOS** | Telegram 6.9+ | WKWebView (Safari/WebKit) |
| **Telegram Desktop (Windows)** | 4.0+ | Chromium (встроенный) |
| **Telegram Desktop (macOS)** | 4.0+ | WebKit (системный) |
| **Telegram Desktop (Linux)** | 4.0+ | Chromium (встроенный) |
| **Telegram Web (browser)** | Любая актуальная версия | Зависит от браузера пользователя |

#### 4.1.2. Режим браузерного SPA

Приложение открывается по прямому URL как обычный веб-сайт:

| Браузер | Минимальная версия |
|---------|--------------------|
| Google Chrome | 70+ |
| Mozilla Firefox | 78+ |
| Safari | 14+ |
| Microsoft Edge (Chromium) | 79+ |
| Samsung Internet | 12+ |

#### 4.1.3. Общие требования к устройству (оба режима)

| Параметр | Требование |
|----------|------------|
| **Экран** | Минимальная ширина 320px. Оптимальная работа от 360px |
| **Интернет** | Стабильное подключение |
| **JavaScript** | Обязательно включён |
| **Cookies / LocalStorage** | Необходимы для хранения JWT-токена |

### 4.2. Требования к серверному окружению

#### 4.2.1. Операционная система сервера

| Параметр | Требование |
|----------|------------|
| **ОС** | Linux (Ubuntu 22.04 LTS / Debian 12 и выше) или любая ОС с поддержкой Docker |
| **Архитектура** | x86_64 (amd64) |

#### 4.2.2. Программное обеспечение сервера

На данном этапе предполагается размещение всей инфраструктуры на единственном сервере. В таблице перечислено как прикладное ПО, так и внешние инфраструктурные зависимости для мониторинга и сбора метрик.

**Прикладное ПО (контейнеры приложения):**

| Компонент | Версия | Описание |
|-----------|--------|----------|
| **Docker** | 24.0+ | Контейнеризация всех сервисов |
| **Docker Compose** | 2.20+ | Оркестрация контейнеров |
| **.NET Runtime** | 8.0+ (внутри контейнера) | Среда выполнения Backend |
| **Node.js** | 20 LTS+ (внутри контейнера) | Сборка Frontend |
| **PostgreSQL** | 15+ | Основная СУБД |
| **MinIO** | RELEASE.2024-01-01+ | S3-совместимое хранилище изображений |
| **Nginx** | 1.24+ | Reverse proxy, TLS-терминация, раздача статики Frontend |

**Инфраструктурное ПО (мониторинг, метрики, логирование):**

| Компонент | Версия | Описание |
|-----------|--------|----------|
| **Prometheus** | 2.50+ | Сбор и хранение метрик (scrape эндпоинтов /metrics) |
| **Grafana** | 10.0+ | Визуализация метрик, дашборды, алертинг |
| **Node Exporter** | 1.7+ | Экспорт системных метрик хоста (CPU, RAM, диск, сеть) в формате Prometheus |
| **cAdvisor** | 0.47+ | Экспорт метрик Docker-контейнеров (потребление ресурсов по каждому сервису) |
| **Loki** | 2.9+ | Агрегация и хранение логов (получает логи через Docker logging driver или Promtail) |
| **Promtail** | 2.9+ | Агент сбора логов из контейнеров, отправка в Loki |

### 4.3. Сетевые требования

| Требование | Описание |
|------------|----------|
| **HTTPS** | Обязателен. Telegram Mini App не запускается без TLS. Необходим валидный SSL-сертификат |
| **Порты** | 443 (HTTPS) -- обязателен для внешнего доступа. 80 (HTTP) -- для редиректа на HTTPS. Остальные порты (5432 PostgreSQL, 9000 MinIO) не должны быть открыты извне |
| **Telegram Bot Token** | Необходим зарегистрированный бот в @BotFather с настроенным Mini App |

### 4.4. Требования к данным и резервному копированию

| Требование | Описание |
|------------|----------|
| **Бэкап БД** | Требуется ежедневное резервное копирование PostgreSQL (pg_dump) с хранением последних 7 копий |
| **Бэкап изображений** | S3-хранилище должно иметь собственную стратегию резервирования (репликация бакетов или периодический snapshot) |
| **Логирование** | Backend должен писать структурированные логи. Централизованный сбор (Loki) |
| **Мониторинг** | Health check эндпоинт (`GET /health`) и мониторинг через Prometheus + Grafana |
